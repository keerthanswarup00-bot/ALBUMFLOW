# Roadmap — albumflow

## Completed

- User authentication (Supabase Auth)
- Studio creation & management
- Album CRUD with image upload
- Three image variants (thumbnail/medium/original)
- Client share links with secure tokens
- Album viewer with page-flip interaction
- Pin feedback system
- Voice message recording for feedback
- Review cycle management (approve/reject/request)
- Multi-studio support with branding
- Dark mode
- Privacy policy, terms, cookie policy pages

## In Progress

- Security audit remediation
- Mobile responsiveness improvements
- Accessibility improvements (WCAG AA target)

## Planned

- Payment/Subscription integration
- Email notifications (Resend/SendGrid)
- Team collaboration (multiple designers per studio)
- Album templates
- Batch operations
- Advanced analytics dashboard
- PDF export/print-ready
- Public gallery/shareable portfolio

## Future Ideas

- AI-powered album layout suggestions
- Client mobile app
- Multi-language support
- Integrations (Lightroom, Pic-Time, etc.)
- API for third-party integrations
